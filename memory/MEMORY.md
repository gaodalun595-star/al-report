# AI Skills & Capabilities

## Core Skills
- **summarize**: Ability to condense long content into concise summaries, focusing on goals, constraints, completed tasks, and next steps.

## Available Skills (On-demand)
- **class-helper**: Designed to help students break down complex problems into actionable steps. (Requires reading `skills\class-helper\SKILL.md` for operational instructions).

## Operational Constraints
- **Skill Installation**: Cannot execute `npx` or external installation commands. New skills must be added by placing their files (e.g., `SKILL.md`) directly into the `skills\` directory within the workspace.
- **Skill Activation**: For new skills to be usable, their files must be manually uploaded/placed in the `skills\` directory so they can be accessed via `read_file`.
